# LABD - Live Apartment Billing Dashboard

A professional, responsive web application for live monitoring of electricity meters with real-time billing and payment integration.

## Features

- **Live Meter Monitoring**: Real-time updates for prepaid and postpaid electricity meters
- **Payment Integration**: Support for bKash, Nagad, Upay, and card payments
- **Wi-Fi Bill Management**: Track and pay ISP bills
- **Responsive Design**: Works perfectly on all devices
- **Real-time Updates**: Automatic data refresh every 60 seconds
- **Professional UI**: Modern, clean interface with smooth animations
